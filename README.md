# My custom Corne Layout

this is my custom corne keyboard layout, it is still a work in progress.


## Layers


- Base Layer 

base qwerty layer.

```
/* BASE layer
 * ,-----------------------------------.   ,-----------------------------------.
 * | tab |  q  |  w  |  e  |  r  |  t  |   |  y  |  u  |  i  |  o  |  p  | bck |
 * |-----+-----+-----+-----+-----+-----|   |-----+-----+-----+-----+-----+-----|
 * | sft |  a  |  s  |  d  |  f  |  g  |   |  h  |  j  |  k  |  l  |  ;  |  '  |
 * |-----+-----+-----+-----+-----+-----|   |-----+-----+-----+-----+-----+-----|
 * | ctl |  z  |  x  |  c  |  v  |  b  |   |  n  |  m  |  ,  |  .  |  /  | esc |
 * `-----------------+-----+-----+-----|   |-----+-----+-----+-----------------'
 *                   | gui | fn1 | ent |   | spc | fn2 | alt |
 *                   `-----+-----+-----'   `-----+-----+-----'
 */
 ```

 - Move Layer 

 a layer dedicated to moving through pages, both with vin-like commands on the right and a mouse emulation on the left side.

```
/* MOVE layer
* ,-----------------------------------.   ,-----------------------------------.
* | tab |  1  |  2  |  3  |  4  |  5  |   |  6  |  7  |  8  |  9  |  0  | del |
* |-----+-----+-----+-----+-----+-----|   |-----+-----+-----+-----+-----+-----|
* | sft |     |     |  ↑  |     |  W↑ |   |  ←  |  ↓  |  ↑  |  →  |     |     |
* |-----+-----+-----+-----+-----+-----|   |-----+-----+-----+-----+-----+-----|
* | ctl |     |  ←  |  ↓  |  →  |  W↓ |   | ms1 | ms2 |     |     |     | esc |
* `-----------------+-----+-----+-----|   |-----+-----+-----+-----------------'
*                   | gui | fn1 | ent |   | spc | fn2 | alt |
*                   `-----+-----+-----'   `-----+-----+-----'
*/
 ```

 - Special Layer

 layer for special characters, including the accents and the ç.

 ```
 /* SPECIAL layer
* ,-----------------------------------.   ,-----------------------------------.
* | tab |  !  |  @  |  #  |  $  |  %  |   |  ^  |  &  |  *  |  (  |  )  | bck |
* |-----+-----+-----+-----+-----+-----|   |-----+-----+-----+-----+-----+-----|
* | sft |  ~  |  '  |  ^  |     |     |   |  -  |  =  |  [  |  ]  |  \  |  `  |
* |-----+-----+-----+-----+-----+-----|   |-----+-----+-----+-----+-----+-----|
* | ctl |     |     |  ç  |     |     |   |  _  |  +  |  {  |  }  |  |  |  ~  |
* `-----------------+-----+-----+-----|   |-----+-----+-----+-----------------'
*                   | gui | fn1 | ent |   | spc | fn2 | alt |
*                   `-----+-----+-----'   `-----+-----+-----'
*/
```

- Config Layer

Layer for rgb and media configurations, and also some mappings for extra layers.

```
/* CONFIG layer
* ,------------------------------------------------------.   ,-----------------------------------------.
* |  boot  | rgb_pri |           |           |     |     |   |        |       |      |     |     |     |
* |--------+---------+-----------+-----------+-----+-----|   |--------+-------+------+-----+-----+-----|
* | rgb_tg | rgb_h_+ |  rgb_s_+  |  rgb_b_+  |     |     |   |  vol+  |  nxt  | ply  |     |     |     |
* |--------+---------+-----------+-----------+-----+-----|   |--------+-------+------+-----+-----+-----|
* | rgb_md | rgb_h_- |  rgb_s_-  |  rgb_b__  |     |     |   |  vol-  |  prv  |      |     |     | UNT |
* `------------------------------+-----------+-----+-----|   |--------+-------+------+-----------------'
*                                |    gui    | fn1 | ent |   |  spc   |  fn2  |  alt |
*                                `-----------+-----+-----'   `--------+-------+------'
*/
```
